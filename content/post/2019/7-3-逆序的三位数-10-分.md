程序每次读入一个正3位数，然后输出按位逆序的数字。注意：当输入的数字含有结尾的0时，输出不应带有前导的0。比如输入700，输出应该是7。

### 输入格式：

每个测试是一个3位的正整数。

### 输出格式：

输出按位逆序的数。

### 输入样例：

```
123
```

### 输出样例：

```
321
```

```
#include<stdio.h>
int main()
{
	int count;
	scanf("%d",&count);
	for(int i=0;i<10;i++)
	{if(count%10!=0)
	printf("%d",count%10);
	count=count/10;
	}
	return 0;
}
```
