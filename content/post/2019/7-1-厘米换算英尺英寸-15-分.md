如果已知英制长度的英尺_f__o__o__t_和英寸_i__n__c__h_的值，那么对应的米是(_f__o__o__t_+_i__n__c__h_/12)×0.3048。现在，如果用户输入的是厘米数，那么对应英制长度的英尺和英寸是多少呢？别忘了1英尺等于12英寸。

### 输入格式：

输入在一行中给出1个正整数，单位是厘米。

### 输出格式：

在一行中输出这个厘米数对应英制长度的英尺和英寸的整数值，中间用空格分开。

### 输入样例：

```
170
```

### 输出样例：

```
5 6





```

```
#include<stdio.h>
int main (void)
	{
		double m,add;
		int foot,inch;
		scanf("%lf",&m);
		m=m/100;
		add=m/0.3048;
		foot=int(add);
		inch=(add-foot)*12;
		printf("%d %d",foot,inch);
		return 0; 
	}
```
